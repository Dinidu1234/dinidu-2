# KickBlastStudentUI

## Requirements
- Windows 10/11
- Visual Studio 2022 (17.8 or later)
- .NET 8 SDK

## How to open in Visual Studio
1. Open Visual Studio.
2. Click **Open a project or solution**.
3. Select `KickBlastStudentUI.sln`.

## How to run
1. Ensure `KickBlastStudentUI` is the startup project.
2. Press `F5` or click **Start**.
3. Login window appears first.

## Default login
- Username: `rashii`
- Password: `123456`

## Change pricing
1. Open **Settings** page after login.
2. Edit values.
3. Click **Save Pricing** to update `appsettings.json` and reload pricing.

## Reset database
Delete this file and run app again:
- `Data/kickblast_student.db`
